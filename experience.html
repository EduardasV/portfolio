Selfkkl-driving car simulator

This simulator has an agent which learned to drive around a town which follows a simple GPS (Global Positioning System) making turns where required using end-to-end machine learning making it train better without having a lot of data. The car would also try to avoid collision with objects it is driving towards by stopping. For example, if there is a blockade within the path of the vehicle it will stop. If the obstacle is no longer there the vehicle will continue to drive.
